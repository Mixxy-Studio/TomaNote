# v0.4.0 - Nueva Experiencia de Escritura

---

## Nuevas Funcionalidades

- **Menú Flotante (Floating Menu)**
  Les contare una historia: hace un tiempo hicimos un menú contextual al oprimir clic derecho, pero según algunas encuestas, muchos usuarios no lo conocían. Así que se decidió crear un menú flotante que siempre está visible, con las mismas funciones del menú contextual (y este último sigue ahí, no te preocupes). Ahora es mucho más accesible para todos.
- **Arrastrar y Soltar tus Pestañas**
  ¡Ordena tus notas a tu manera! Ahora puedes arrastrar y soltar las pestañas para ponerlas en el orden que tú quieras. Pensamos en quienes necesitan tener sus notas organizadas de forma personalizada.
- **Selector de Tamaño de Fuente**
  Hola familia, sé que lo pedían con el corazón y pues... ¡aquí está! Ahora puedes elegir entre tres tamaños de texto: Compacto, Normal y Grande. El propósito de TomaNote es mejorar la accesibilidad para todos. (Incluida mi abuelita, abuelita te quiero).
- **Tu Propia Fuente con Google Fonts**
  Porque sabemos que cada quien tiene su estilo, ahora puedes usar tu fuente favorita directamente desde Google Fonts. Solo copies la URL, la pegas en Settings y listo.
- **Emojis en Pestañas Fijadas**
  Cuando fijas una pestaña, ahora le asignamos un emoji aleatorio para identificarla más rápido. Tú también puedes elegirle el emoji que más te gusta.
- **Cierra tus Pestañas con ESC**
  Atajo de teclado: ahora puedes cerrar la pestaña activa simplemente oprimiento la tecla ESC (disponible en escritorio).
- **Botón de Settings en el Menú Flotante**
  El botón de configuración ahora está integrado en el menú flotante, más intuitivo y al alcance de todos.
- **Modal de Settings Renovado**
  El modal de información y configuración se renovó completamente. Ahora tiene secciones específicas: Acerca de, Licencia, Términos, Roadmap, Tipografía, Temas e Idioma.

## Mejoras Visuales

- Sistema de animaciones suaves para una experiencia más fluida
- Iconos completamente refactorizados y consistentes (ahora son un componente reutilizable)
- Empty State renovado con un diseño más atractivo y moderno
- Colores y botones más intuitivos para el usuario
- Scrollbar con colores mejorados
- Migración parcial a TailwindCSS para estilos más modernos
- Nueva paleta de colores para los temas existentes

---

## Mejoras Técnicas

- Tests Unitarios: Agregamos pruebas automáticas para componentes core (tabs, floating menu, keyboard shortcuts, drag & drop). Ahora el código es más robusto.
- Sistema de eventos mejorado (tabsChanged) para una mejor comunicación entre componentes
- Integración de SortableJS para el arrastre de pestañas
- Refactorización completa del sistema de componentes de iconos
- Mejoras en el rendimiento de carga
- Archivos de caché de Astro ahora ignorados en git

## Arreglos (Bug Fixes)

- El menú contextual ahora está deshabilitado en dispositivos móviles
- Se corrigió la selección de texto al hacer clic en los botones de herramientas

- Las pestañas fijadas muestran el emoji correctamente
- Los botones del menú flotante ya no desaparecen cuando el menú está abierto
- El botón de "Nueva Pestaña" ahora tiene un comportamiento más predecible
