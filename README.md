## 📋 Description

TomaNote is a modern, minimalist notepad that runs directly in your browser. No accounts, no sign-up, no servers. Just open it and start writing. All your notes are saved locally on your device — private by design, offline by default.

> Want to see what's coming next?
> Check the **Roadmap** inside the app at [tomanote.app](https://tomanote.app) ✨

## 🚀 What's new — v0.4.2

This release focuses on deprecation handling and mobile/PWA improvements.

### 🔔 Modal confirmation for tab deletion
Replaced browser's native confirm() dialog with a custom modal. The native confirm() will be deprecated in future browser versions.

### 📱 Floating menu positioning fix
Improved positioning for mobile browsers and PWA installations. The menu now correctly adapts to:
- Safe area insets on iOS
- Navigation bars on Android browsers
- Virtual keyboard detection

### 🧪 Test coverage
Added 19 new tests for the new modules. All 163 tests passing.

## 📦 Installation

### Use it online

1. Go to [tomanote.app](https://tomanote.app)
2. Start writing.

### Run it locally

```bash
# Clone the repo
git clone https://github.com/Mixxy-Studio/TomaNote.git

# Install dependencies
npm install

# Start dev server
npm run dev
```

## ⚠️ Important

- Notes are stored in your browser's LocalStorage
- Clearing your browser cache will delete your notes — back them up regularly
- No data is ever sent to external servers

## 🔒 Security

- Don't store sensitive information
- Data lives locally on your device
- No external data transmission
